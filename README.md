# Job_Predict
对Boss直聘的海量IT招聘信息进行数据分析，针对关键特征建立职业预测模型，并基于Python+Django和微信小程序云开发搭建系统


1.数据来源
Boss直聘网站按IT职业分类进行搜索，使用网络爬虫获取网页的关键信息。

  Python支持库

        Pandas 1.5.3
        
  火狐浏览器 113.0.2（32位）
